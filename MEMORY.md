# MEMORY.md - Ledger's Long-Term Memory

## Commission Report Process

### Platform Fees
- Viator: 30%
- Airbnb: 25%
- GetYourGuide: 25%
- Bokun (Direct): 3%

### Math Calculations
- Viator only shows net → calculate gross by reversing the 30%
- We pay bar based on **net** (not gross)
- Gross is just for our internal tracking

### Required Data Per Booking
- Booking date
- Activity date
- Gross rate
- Net rate
- Number of guests

### Commission Calculation
- 10% of net = bar's commission
- If total sales < $15k → apply $1,500 minimum
- Add liquor costs
- Total owed to bar

### Monthly Process
1. Get data from: Viator, GetYourGuide, Airbnb, Bokun (direct)
2. Calculate gross from net where needed
3. Sum all net revenue
4. Calculate 10% commission
5. Apply $1,500 minimum if under $15k
6. Add liquor costs
7. Create report in Google Sheets
8. Send to bar owner
